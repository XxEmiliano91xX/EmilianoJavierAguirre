¿Qué es HTML y CSS?

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido en la web.

CSS

CSS (Cascading Style Sheets) es el lenguaje que define el estilo visual de una página web, incluyendo colores, fuentes y disposición.
